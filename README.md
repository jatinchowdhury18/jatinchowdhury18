## Morning 👋

**Jatin Chowdhury is an audio signal processing engineer**

### Things I'm doing:

- Developing libraries and tools for audio developers, including:
  - [`chowdsp_utils`](https://github.com/Chowdhury-DSP/chowdsp_utils/tree/master/modules): A library of C++, DSP, and other utilities for developing audio plugins.
  - [RTNeural](https://github.com/jatinchowdhury18/RTNeural): A neural network inference engine for real-time applications.
  - [`chowdsp_wdf`](https://github.com/Chowdhury-DSP/chowdsp_wdf): A library for developing circuit simulations using Wave Digital Filters.
- Creating audio effect plugins as [Chowdhury DSP](https://chowdsp.com), including [ChowTapeModel](https://github.com/jatinchowdhury18/AnalogTapeModel), [BYOD](https://github.com/Chowdhury-DSP/BYOD), and [ChowMatrix](https://github.com/Chowdhury-DSP/ChowMatrix).
- Pursuing a PhD in Electrical Engineering and Music Technology at MIT.
- Writing about audio signal processing and programming on [Medium](https://jatinchowdhury18.medium.com).
- Giving [talks](https://ccrma.stanford.edu/~jatin/slides/) and workshops on real-time audio programming and performance engineering.
- Making music!

For more, see my [portfolio](https://ccrma.stanford.edu/~jatin/portfolio.html), [research page](https://ccrma.stanford.edu/~jatin/research.html), and [personal website](https://ccrma.stanford.edu/~jatin).
